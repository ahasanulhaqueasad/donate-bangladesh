# 🇧🇩 Donate Bangladesh - Flood Relief & Humanitarian Aid Platform

A modern, interactive, and responsive donation platform designed to support flood victims in Noakhali, Feni, and injured individuals in the Quota Movement. Built with **HTML5**, **Tailwind CSS**, and **DaisyUI**, this project focuses on seamless user experience and real-time transaction tracking.

## ✨ Key Features

- **Real-time Balance Update:** Your personal fund decreases and the specific sector's fund increases instantly upon donation.
- **Dynamic Donation History:** Every donation is logged with the specific amount, location/cause, and the exact date/time.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
- **Interactive UI:**
  - Sticky Navigation with glassmorphism effect.
  - Success Confirmation Modals.
  - Smooth toggling between Donation and History sections.
- **Insightful Blog:** A dedicated section with 10 informative articles regarding humanitarian aid.

## 🛠️ Tech Stack Used

* **Frontend:** HTML5, Tailwind CSS
* **UI Components:** [DaisyUI](https://daisyui.com/)
* **Interactivity:** Vanilla JavaScript (ES6+)
* **Icons & Images:** Unsplash API & Custom Assets
